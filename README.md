# Hiragana Trainer – PWA (GitHub Pages ready)

Dieser Ordner ist bereit für GitHub Pages. Einfach alle Dateien ins Repo hochladen und Pages aktivieren.

## Dateien
- index.html, style.css, app-inline.js
- manifest.webmanifest, sw.js
- icon-192.png, icon-512.png

## GitHub Pages
1) Neues Repo anlegen (public).
2) Alle Dateien hochladen und committen.
3) Settings → Pages → Source: Deploy from a branch, Branch: main, Folder: /(root). Speichern.
4) Seite öffnen (z. B. https://<user>.github.io/<repo>/) und über das Installieren-Icon als App hinzufügen.

Hinweis: start_url und scope sind relativ (./), daher funktioniert die PWA auch unter einem Subpfad (GitHub Pages).
